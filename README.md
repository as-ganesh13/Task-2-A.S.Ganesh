Expenses Tracker 

A simple and efficient Expense Tracker built with Python that helps users manage daily expenses. The application stores expense records in a CSV file and provides features for tracking, searching, and analyzing spending habits.

Features

Add new expenses
View all expenses
Calculate total expenses
Search expenses by category
Delete expense records
Automatic date tracking
CSV-based data storage
User-friendly menu-driven interface

Technologies Used

Python
CSV File Handling
Functions
Loops
Conditional Statements
Lists and Dictionaries

Sample Categories

Food
Travel
Shopping
Entertainment
Education
Health

Learning Outcomes

This project demonstrates:
Data accumulation using variables
File handling with CSV
CRUD operations (Create, Read, Update, Delete)
Modular programming using functions
Real-world data management concepts
